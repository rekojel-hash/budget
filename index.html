📤 Exporter les données
Téléchargez une copie complète de toutes vos données (transactions, objectifs, paramètres) au format JSON. Conservez ce fichier précieusement pour ne rien perdre.
ℹ️ Format JSON Contient 100% de vos données : toutes les transactions sur toutes les périodes, vos objectifs d'épargne et la configuration. Réimportable à tout moment.
🏦 Importer un relevé bancaire
Importez directement vos opérations converties depuis le PDF de votre banque. L'application lira automatiquement les dates, débits et crédits.
📥 Importer des données
Restaurez vos données depuis un fichier JSON exporté précédemment. Vous pouvez choisir de fusionner avec les données existantes ou de les remplacer entièrement.
⚠️ Attention En mode "Remplacer", toutes vos données actuelles seront effacées avant l'import. En mode "Fusionner", les doublons sont détectés par identifiant unique.
🗂️ Sauvegardes rapides
Créez jusqu'à 5 instantanés nommés de vos données dans le navigateur. Utile pour tester des scénarios ou conserver des états clés sans télécharger un fichier.
🗑️ Réinitialisation
Supprime définitivement toutes les transactions et objectifs. Cette action est irréversible. Pensez à exporter vos données avant.
